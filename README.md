# DJSCC for Wireless Image Transmission

Implementation of [Deep Joint Source-Channel Coding for Wireless Image Transmission](https://ieeexplore.ieee.org/document/8723589)

### Deep JSCC-Q Web Interface
This repository contains a modern Flask + Keras 3 implementation for testing the model via a dynamic web interface.

To run the inference server locally:
```bash
cd Keras_Implementation
pip install -r requirements.txt
python app.py
```
Then open `http://localhost:5000` in your browser.
